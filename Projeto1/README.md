# Gerador de Lista de Arquivos
Este projeto em Python cria automaticamente um arquivo de texto (.txt) contendo o nome de todos os arquivos presentes em uma pasta ou diretório escolhido pelo usuário.

A ideia surgiu da necessidade de listar nomes de arquivos em ambientes onde o sistema operacional (como o Windows) não oferece uma forma simples de copiar e colar os nomes diretamente.

## Como usar
1. Clone o repositório: git clone https://github.com/Graciele-NL/Organizar-arquivos;
2. Acesse a pasta do projeto: cd Organizar-arquivos;
3. Execute o script: python src/NomesArquivos.py;
4. Informe o caminho da pasta quando solicitado.

O arquivo .txt será criado no mesmo diretório informado.

## Exemplo de saída
arquivo1.pdf
arquivo2.docx
imagem.png
